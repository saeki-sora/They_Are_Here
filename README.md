# They Are Here

3D一人称視点のステルスホラーゲームです。  
迷路状のマップでカギを拾い、ゴールへ辿り着くことを目指しますが、個性の異なる複数の敵AIが追跡してきます。

## 技術的な工夫

### 敵AI（ステートマシン）
- **索敵（Search）**：巡回しながらプレイヤーを探す
- **追跡（Chase）**：A\*経路探索でプレイヤーを追う
- **見失い（Lost）**：最後に目撃した位置に向かう
- **ストーカー**：追跡速度が高く、迂回ルートで回り込む行動をとる
- **ラバーズ（ペア）**：2体が連携して挟み撃ちを狙う

### A\*経路探索（Pathfinder）
- マップをグリッドに分割し、プレイヤーへの最短ルートを動的に計算
- フレームごとの計算回数を上限管理し、フレームレートへの影響を最小化

### マップ自動生成（MakeMap）
- 難易度選択時にタイトル画面のバックグラウンドスレッドで全難易度分のマップを並列生成
- ゲーム開始時に待ちが発生しないよう先読み生成

### スタミナとゾーン設計
- ゴールに近いほどスタミナ回復速度が落ちる仕組みにより、終盤の緊張感を維持

### マネージャーパターン
- `SceneManager` / `CollisionManager` / `EffectManager` / `SoundManager` / `DifficultyManager` など各責務を独立したマネージャークラスに分離

## ビルド手順

1. Visual Studio 2022 をインストール（C++ ゲーム開発ワークロードを選択）
2. `TheyAreHere.sln` を開く
3. ビルド構成を `Release` / `x64` に変更
4. **Ctrl + Shift + B** でビルド → `x64/Release/` に実行ファイルが生成されます

> **注意：** `assets/` フォルダ（モデル・テクスチャ・サウンド）はリポジトリに含まれていません。  
> 実行には別途 assets フォルダを `src/` と同じ階層に配置してください。
